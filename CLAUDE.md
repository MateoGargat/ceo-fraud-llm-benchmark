# Project Rules

## Git Commits
- **Language:** English only for commit messages
- **Format:** Conventional commits (`feat:`, `fix:`, `docs:`, `refactor:`, `test:`, `chore:`)
- **No AI signatures:** Never add `Co-Authored-By` lines referencing Claude, Anthropic, OpenAI, or any AI model
- **Max 72 chars** on the first line
- Commit messages should be concise and describe the "why", not the "what"

## Code
- Python 3.12+
- Tests with pytest, TDD approach
- All code and comments in English
